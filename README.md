# Monogram

A minimal, analog-inspired journaling and curation platform built with React, TypeScript, and Vite.

## Features

- **Terminal-Style Selectors**: Minimal CLI-inspired dropdowns with typing functionality
- **Analog Minimalism**: Clean, calm interface with retro terminal aesthetics
- **Space-Based Organization**: Organize content into curated spaces
- **Responsive Dashboard**: Modern stats and overview cards
- **Type-Safe**: Built entirely with TypeScript

## Tech Stack

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Smooth animations
- **React Router** - Client-side routing
- **Lucide React** - Icon library

## Getting Started

Install dependencies:
```bash
npm install
```

Start the development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

## Project Structure

```
src/
├── components/       # Reusable UI components
│   ├── ui/          # Base UI primitives (buttons, cards, selectors)
│   ├── layouts/     # Layout components
│   └── ...          # Feature components
├── pages/           # Page-level components
├── contexts/        # React contexts (auth, etc)
├── styles/          # Global styles
└── main.tsx         # Entry point
```

## Design Philosophy

Monogram embraces **analog minimalism** — interfaces that feel tactile and intentional, inspired by early computer terminals and typewriter consoles. Every interaction should feel like a keystroke, not a click.
